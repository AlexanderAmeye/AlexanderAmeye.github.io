Stylized Water Shader

A stylized water shader created in the Unity3d game engine designed to work with the new scriptable rendering pipelines. The shader has depth-based colors, horizon colors, color grain, transparency, surface foam, intersection foam, foam distortion, refraction, caustics, reflections and waves. It uses a custom inspector to edit the parameters in an organized way. The shader works in perspective mode as well as orthographic mode and supports all the major platforms (Windows, Mac, Linux, Android, iOS, PS4, Xbox, Nintendo Switch, VR).

January 2020

Workmate

For my OOP course I designed and created an employee polling system. Employees answer questions related to their work-life asked by their employers. By doing this they receive company currency which they can use for various company related beneftis. Employers have an easy way of gaining insight into how content their workers are and where they can improve. Company administrators have access to a dashboard website where they can analyze answered questions and various statistics about the results.

For this project an android application was created within android studio as well as a dashboard website. The application and the website communicate together through an SQL server.

Spring 2019
