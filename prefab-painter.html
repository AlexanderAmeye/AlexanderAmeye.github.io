Prefab Painter

I created a free prefab painter. It has some cool features like a prefab list that resizes dynamically, customizable brush sizes and densitites, layer-restriction, random sizes and rotations etc. I like working on it but since the release of Unity's free Polybrush, I'm kind of hesitant to put too much work into it.



You can download the project here. It was updated to work with Unity 2019.
