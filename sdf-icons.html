Introduction

After learning about signed distance functions, I wanted a fun way to practice using them in a shader. I decided to create some icons using SDFs. The nice thing about these is that you don't need to sample any textures and the icons stay perfectly crisp even when zooming in.


Signed Distance Functions

I created custom function nodes to generate different SDF types like circles, stars, vesicas, polygons etc. I then also made custom functions that can sample an SDF and blend two SDFs together.


Pokémon Type Icons

Using these SDF nodes I started recreating the Pokémon type icons.




Celestial Bodies

Other than the Pokémon icons, I also created some icons of celestial bodies. These were simple enough shapes but fun to create.
