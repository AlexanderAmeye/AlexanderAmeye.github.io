In this tutorial I will be explaining how you can create a tessellated plane in Unity. A tessellated plane is a regular plane that you add more vertices to. We need more vertices than the standard Unity plane for our next tuturial where we will be adding wave displacement. For this discplacement to be smooth, we need a good amount of vertices.
